Grand Canyon Landing Page

This is a simple and responsive landing page made with React, Tailwind CSS, and AOS animations.
It is designed to show a modern web layout with smooth scroll effects and a clean design.

Features:
- Fully responsive for all devices
- Smooth animations on scroll
- Organized and reusable components

Technologies Used:
- React.js
- Tailwind CSS
- AOS (Animate on Scroll)

How to Run:
1. Download or clone this project.
2. Open it in VS Code.
3. Open a terminal and run:
   npm install
4. Then run:
   npm start
5. The project will open on your browser at http://localhost:3000

Folder Structure:
- public: static files like index.html and images
- src: contains all components and pages
- package.json: project configuration
